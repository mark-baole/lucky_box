## Quick Start

1.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the app**:
    ```bash
    python -m uvicorn backend.main:app --reload
    ```

3.  **Play**:
    Open http://localhost:8000 in your browser.
